# 🚀 SpeedTrack Pro  
**Real-time Speed Monitoring & Alert System**  

SpeedTrack Pro is a **React.js & TypeScript-based speed monitoring app** that tracks real-time speed using GPS and alerts users when exceeding speed limits. It integrates with **Google Maps API** for accurate navigation, ensuring a safer driving experience.  

---

## 📌 Features  
✅ **Real-Time Speed Tracking** – Uses GPS to monitor speed dynamically.  
✅ **Speed Limit Alerts** – Sends instant notifications when overspeeding.  
✅ **Google Maps Integration** – Displays routes and real-time speed data.  
✅ **Custom Speed Warnings** – Users can set personal speed limits.  
✅ **Dark & Light Mode** – User-friendly UI for different lighting conditions.  

---

## 🛠️ Built With  
- **React.js** – Frontend framework  
- **TypeScript** – Strongly typed JavaScript  
- **Google Maps API** – Navigation & speed tracking  
- **Tailwind CSS** – Modern UI styling  
- **React Hooks** – `useState`, `useEffect`, `useContext`
